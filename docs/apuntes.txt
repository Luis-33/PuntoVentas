LEVANTAR EL PROYECTO 
Instalar 
    -docker
    -visual studio code
    -git bash
    -Postman
    -Node js
Bajar el respositorio
    -git clone (el url del repositorio)

instalar dependencias
npm init -y // en caso de inicio de proyecto desde 0 
npm install express pg dotenv cors morgan 
npm install nodemon --save-dev
    EXPLICACION DEPENDENCIAS
//npm init -y: para que configure el archivo package.json
// instalacion de dependencias clave
    Express: crea servidor http
    pg: para la Conexión PostgreSQL
    dotenv: para variables de entorno
    Cors: Permite que el frontend consuma tu API 
    Morgan: Muestra en consola cada request
    nodemon --save-dev: reinicia el servidor automaticamente
todo se ejecuta en la carpeta backend 

Levantar docker compse
docker compose up -d 
// esto sirve pra levantar los servicios programados en el archivo docker-compose.yml
// pero primero deberás de tener ejecutada la aplicacion docker desktop
//y deberás estar posicionado al nivel del archivo docker-compose.yml para que funcione
// una vez levantado todo, entras a al pgadmin con las credenciales correspondientes
//click derecho en servers luego register/server y aplicas la siguiente configuración
//pestaña general 
//opcion name = dockerconnection
//pestaña connection 
// opcion host name/addres= postgres
// port =5432, maintenance database=ventas_db, username= root, password= root
// y guardar
// despues importas las tablas en la base de datos ventas_db
